# 📌 Kanban Board

A simple and interactive **Kanban Board** built using **HTML, CSS, and JavaScript** to manage tasks efficiently with drag-and-drop functionality.

---

## 🚀 Features

- ➕ Add tasks dynamically  
- 🖱 Drag and drop tasks between columns  
- 📦 Organize workflow (To Do → In Progress → Done)  
- 🎯 Clean and minimal UI  
- ⚡ Pure JavaScript (no libraries)

---

## 🛠 Tech Stack

- HTML  
- CSS  
- JavaScript (DOM + Drag & Drop API)

---

## 📂 Project Structure
kanban/
│── index.html
│── style.css
│── main.js 
---

## 🎮 How It Works

1. Click **"Add Task"**  
2. Enter your task  
3. Drag the task  
4. Drop it into any column  

---

## 🧠 Concepts Used

- DOM Manipulation  
- Event Listeners  
- Event Delegation  
- Drag and Drop API  
- Dynamic Element Creation
